## Installation
Run a Container Docker at Docker Desktop and start it: 
    
    - docker-compose up --build
    - docker-compose logs -f <service_name>
    - docker-compose up -d
    - docker-compose up

    - npx prisma migrate dev --schema=./backend/prisma/schema.prisma   
    
    - docker-compose up -d 

Open a new terminal and type:

    - npx prisma studio
